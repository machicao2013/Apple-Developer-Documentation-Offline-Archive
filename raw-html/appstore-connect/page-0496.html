Test a beta version

View build status and metrics

Required role: Account Holder, Admin, App Manager, Developer, or Marketing. View role permissions.

  1. In Apps, select the app you want to view.

  2. Click the TestFlight tab.

  3. In the sidebar, under Builds, click the platform.

  4. In the table on the right, view the build status and metrics for versions and build groups.

    Data column

    Description

    Status

    View the status of a build.

    Groups

    View the tester groups that are testing a build.

    Invites

    View the number of invited testers.

    Installs

    View the number of installations on devices.

    Sessions

    View the total number of sessions in the last seven days across all testers.

    Crashes

    View the number of times a build crashed on tester devices.

    Feedback

    View the number of feedback submissions you recieved from your testers.

TestFlight testers automatically share crash logs to developers. To view crash reports directly in Xcode for apps distributed using TestFlight, click Open in Xcode next to each crash feedback entry on the Crash Feedback page within the TestFlight tab.
