Set up OS data transfer

OS data transfer allows users to seamlessly transfer your app’s data across Android and iOS by adding app IDs from your Android app in App Store Connect. During a data transfer, the user’s Android device will provide iOS a list of installed app IDs, which will be matched against what you have provided. This helps ensure that apps in the mapping table you provide are migrated.

In order to support this, you must provide your app information from Android so that we can correctly install your app. This includes your Android Package Name and the SHA-256 certificate fingerprint of your Signing Key Certificate.

You can also enable data migration by implementing AppMigrationKit.

Each iOS app can only map to one Android app from Google Play; however, you can add up to 5 app ID mappings per app.

Note: Users can only use OS data transfer for your app if you set it up both in App Store Connect and with Google. Additionally, users must be running iOS 26.3 or later and a version of Android that also supports OS data transfer.

Required role: Account Holder, Admin, or App Manager. View role permissions.

Set up OS data transfer

  1. In Users and Access, click Integrations.

  2. Under Additional, click OS Data Transfer.

  3. Click Get Started.

    If you’ve already set up an OS data transfer you can click the (+) button.

  4. Select the app you wish to set up data transfer for.

  5. Click Add.

  6. Provide the Android App Package Name for the Android version of your app and the SHA-256 certificate fingerprint.

    Note: You can add up to 5 SHA-256 certificate fingerprints in the event that you change your key.

  7. If you have multiple Android app IDs associated with the same iOS app you can add up to 4 additional app IDs by clicking Add Another Android App at the bottom of the page.

  8. On the top right, click Save.

Edit OS data transfer details

  1. In Users and Access, click Integrations.

  2. Under Additional, click OS Data Transfer.

  3. Click Edit next to the app you wish to edit.

  4. Make the necessary changes and then on the top right, click Save.

Delete OS data transfer App IDs

  1. In Users and Access, click Integrations.

  2. Under Additional, click OS Data Transfer.

  3. Click Edit next to the app you wish to edit.

  4. Click Delete at the bottom of the page.

  5. In the dialog that appears, click Delete.

    You can also delete all of the Android app IDs you have added for the app by clicking Delete All at the bottom of the page.
