Win-back eligibility report

The Win-back eligibility report includes the total aggregated count of subscriptions in a churned state as of the reporting date. The data in the report can be used to estimate the number of users eligible for a win-back offer.

Report Field

Data Type

Notes

App Name

String

Title of your subscription’s parent app.

App Apple ID

BigInt

Apple ID of your subscription’s parent app.

Subscription Name

String

Title of your subscription.

Subscription Apple ID

BigInt

Apple ID of your subscription.

Subscription Group ID

BigInt

Your subscription’s Group ID (formerly Family ID).

Standard Subscription Duration

Int

Duration of the standard subscription: 7 Days, 1 Month, 2 Months, 3 Months, 6 Months, or 1 Year.

Paid Subscription Tenure

Int

Number of months of paid service accumulated prior to cancelation, rounded down to the nearest month.

Time Since Last Subscribed

Int

Number of months since the susbcription churned, rounded down to the nearest month.

Has Previously Redeemed Win-Back Offer

String

Indicates if the user has perviously redeemed a win-back offer. Values in this field appear as Yes or <blank>.

Time Since Last Win-Back Offer

Int

Number of months since the last win-back offer, rounded down to the nearest month.

Territory

String

Two-character ISO country code indicating the App Store country or region for the purchase.

Churned Subscription Count

Int

Number of subscriptions in the churned state. Individual rows will appear only if there are 5 or more churned subscriptions.
