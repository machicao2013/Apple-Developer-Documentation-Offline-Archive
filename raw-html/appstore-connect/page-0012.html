Configure Game Center

Manage groups

A group must contain at least one app. Once you create a group from an app, you can add additional apps to it to share leaderboards, achievements, and challenges data. Learn how to share activities’ data using App Store Connect API.

When you add an app with existing Game Center components to a group, you can choose which of those existing leaderboards, achievements, and challenges to add to the group. Once added, these components are defined in the group as they were in the app prior to being grouped. Combining these Game Center components will replace the version in the app with the one you choose from the group. App Store Connect simplifies this process by presenting only compatible components and available options at each step. The options that App Store Connect presents to you're dependent on your app and group data. Types of assets (for example, leaderboard sets) that the app doesn’t have won't display.

Note:

  • Apps can only belong to one Game Center group. If you wish to add your app to a new group, it must be removed from any existing groups.
  • If you add live leaderboards, achievements, and challenges to a group, they stay live without submitting an app version.

Guidelines for moving achievements

There are no restrictions for adding achievements to a group; however, the following remain true:

  • The maximum number of points for each achievement is 100.

  • The total maximum number of points for a single app's achievements is 1000.

  • There are no restrictions on the total points for all group achievements.

Guidelines for moving challenges

Challenge can only be moved to a group with its associated leaderboard.

Learn how to move activities using App Store Connect API.

Required role: Account Holder, Admin, App Manager, Developer, or Marketing. View role permissions.

Create groups

  1. In Apps, select your app you want to view.

    To create a new Game Center group, select an app that isn’t already part of a group.

  2. In the sidebar, click Game Center.

  3. In the Groups section, select Move to Group.

  4. Select Move to a New Group.

  5. Under the Move to a New Group section, enter your group reference name.

    • If your app doesn’t have existing leaderboards, achievements, or challenges, this is the final step for creating your Game Center group. Click Add to Group.

    • If your app has existing leaderboards, achievements, or challenges, click Next. You can enter a new group ID for each components to add it as a new component to the group in the Move Leaderboards, Move Achievements, or Move Challenges prompt. Their IDs are listed with a “grp.” prefix. You can keep the ID the same, or create a new ID in the New ID field.

  6. Click Next.

  7. Click Add to Group.

Add an app to an existing group

  1. In Apps, select your app you want to view.

  2. In the sidebar, click Game Center.

  3. In the Groups section, select Move to Group.

  4. Under the Move to an Existing Group, select the Game Center group that you’d like to add your app to.

  5. Click Next.

  6. Review the group IDs, and modify as needed, for your leaderboards, achievements, and challenges that are being moved to the group.

    Note:

    • Your existing Game Center components’ IDs are listed with a “grp.” prefix.

    • Add an app’s leaderboard to the leaderboards already defined for the group. The leaderboard appears in the group just as it did for the app.

  7. Click Continue.

  8. Click Add to Group.

Remove an app from a group

An app can only be removed from a group when a previous version of the app hasn’t been published to the App Store with a group leaderboard or achievement attached.

  1. In Apps, navigate to the ellipsis menu (...) at the top left of the page.

  2. Click Game Center Groups.

  3. Select the group you want to edit in the Manage Existing Groups section.

  4. In the Attached Apps section, find the app you want to convert from a group game to a single game.

    • If the app is the only one listed in the group and doesn’t contain live Game Center components, click Delete Group in the top left corner.

    • If there are multiple apps, click the delete button next to the app name. Be sure to remove any multiplayer compatibility settings defined for the app version.

    • Click Save.

Edit groups

After you create and add apps to a group, you can edit the group’s properties from the Game Center groups page. You follow the same steps to create and manage group leaderboards, leaderboard sets, achievements, and challenges as you'd for a non-grouped app.

  1. In Apps, navigate to the ellipsis menu (...) at the top left of the page.

  2. Click Game Center Groups.

  3. Select the group you want to edit in the Manage Existing Groups section.
