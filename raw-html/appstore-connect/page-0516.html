Understanding the Core Technology Fee

Core Technology Fee report fields

Core Technology Fee report fields

Report field

Data type

Notes

Developer Name

String

Developer's legal entity name.

Reporting Month

String

The report is for the month and the year displayed.

Reporting Date Range

String

The report is for the date range printed.

App Name

String

The name of the app that has accumulated first annual installs over the free threshold.

Apple Identifier

Int

Apple ID, a unique identifier automatically generated for your app when you add the app to your account.

Territory

String

The territory where the app was installed. Based on the territory of the user's Apple Account.

First Annual Installs

Int

The first time an app is installed by an account in the EU in a 12-month period.

Fee-Exempt First Annual Installs

Int

Number of free first annual installs.

Fee-Eligible First Annual Installs

Int

Number of first annual installs over the free threshold.

Fee Per Install

Decimal

Monthly fee for each first annual install over the free threshold. The fee is exclusive of any applicable surcharge and taxes.

Core Technology Fee

Decimal

Total monthly fees for all fee-eligible first annual installs. This fee is exclusive of any applicable surcharge and taxes.

Fee Currency

String

Three-character ISO code for the currency in which the fee is represented in.
