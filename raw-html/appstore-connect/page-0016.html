Configure Game Center

Manage leaderboards

Leaderboards help encourage friendly competition within your game. When you configure leaderboards in App Store Connect, you specify details such as the scores to collect and how to order them. For each language you want a leaderboard to display in, you specify the leaderboard name, score format, and score unit. You can also include a localized image to illustrate the score in the leaderboard.

There are two types of leaderboards that can be created:

  • Classic leaderboards: A leaderboard where score data will never reset unless the leaderboard is deleted.

  • Recurring leaderboards: A leaderboard where score data will continually be reset based on set preferences.

The limit to how many leaderboards you can create per app differs based on whether it’s with or without leaderboard sets:

Without Leaderboard sets:

  • Up to 100 leaderboards per app

With Leaderboard sets:

  • Up to 500 leaderboards total per app

    • Maximum 100 leaderboard sets per app

    • Maximum 100 leaderboards per set

Leaderboards must be approved before they are available to players. Learn how to submit Game Center components for review.

Learn how to configure leaderboards with the GameKit.

Required role: Account Holder, Admin, App Manager, Developer, or Marketing. View role permissions.

Create a classic leaderboard

  1. In Apps, select the app you wish to enable for Game Center.

  2. In the sidebar, click Game Center.

  3. Click Add Leaderboard.

    Note: If a leaderboard already exists, click the add button (+) in the Leaderboards section. If a leaderboard set already exists, either create a new leaderboard set or select a set and add your new leaderboard to that set.

  4. Enter the reference name and leaderboard ID.

  5. Select Classic Leaderboard.

  6. Click Next.

  7. Enter the score format information.

  8. Click Create.

  9. Under Leaderboard Localization, add one or more languages by clicking Add Localization.

  10. Enter the localization information.

  11. Click Save.

    To add additional language support for your leaderboards, repeat the above steps for each language.

    When you're ready, submit your leaderboard for review.

Create a recurring leaderboard

Recurring leaderboards add flexibility to your leaderboard configuration by allowing you to reset your leaderboards’ scores. The ability to reset your leaderboards creates a more competitive gaming experience, as it allows players to set a new top score for each leaderboard recurrence.

Leaderboards have a minimum recurrence of five minutes, a maximum recurrence of 30 days, and aren't allowed to overlap.

  1. In Apps, select the app you want to view.

  2. In the sidebar, click Game Center.

  3. Click Add Leaderboard.

    Note: If a leaderboard already exists, click the add button (+) in the Leaderboards section. If a leaderboard set already exists, either create a new leaderboard set or select a set and add your new leaderboard to that set.

  4. Enter the reference name and leaderboard ID.

  5. Click Next.

  6. Select Recurring Leaderboard.

  7. Complete the Start Date and Time, Duration, and Restarts Interval fields.

  8. Click Next.

  9. Enter the score format information.

  10. Click Create.

  11. Under Leaderboard Localization, add one or more languages by clicking Add Localization.

  12. Enter the localization information.

  13. Click Save.

    To add additional language support for your leaderboards, repeat the above steps for each language.

    When you're ready, submit your leaderboard for review.

Set a default leaderboard

  1. In Apps, select the app you want to view.

  2. In the sidebar, click Game Center.

  3. Locate the Default column in the Leaderboards section, then hold the pointer over the leaderboard you'd like to set as default.

  4. Click Make Default.

Arrange leaderboards

Leaderboards are presented to players in the order they appear in App Store Connect. You can change the order of the leaderboards on the app’s Game Center page.

  1. In Apps, select the app you want to view.

  2. In the sidebar, click Game Center.

  3. Click Edit Order on top right.

    To arrange your leaderboards within a leaderboard set, select the leaderboard set that you want to view and click Edit Order.

  4. Click the reordering control on the left side of the Leaderboards row and drag it to a new position.

  5. Click Save.

Archive or unarchive a leaderboard

You can archive live leaderboards to remove them from all relevant sections of Game Center’s user interface. This also ensures they won’t return when retrieving a GKLeaderboard. You can archive and unarchive leaderboards at any time but it may take up to 24 hours to take effect to the players.

  1. In Apps, select the app that you want to manage.

  2. In the sidebar, click Game Center.

  3. In the Leaderboards section, select the leaderboard that you want to archive.

    If leaderboard sets exist, select the leaderboard set that you want to view and select the leaderboard that you want to archive.

  4. Click Archive Leaderboard.

    If the leaderboard is already archived, the Unarchive Leaderboard option will be displayed. Click it and follow the prompt to unarchive it.

  5. Click Archive.

To view the list of archived leaderboards, click the ellipsis menu (…) within the Leaderboards section, and select View archived leaderboard. If there are Leaderboard Sets, select the set you want to view first.

Delete a leaderboard

The status of a leaderboard is displayed in the Status column of the Leaderboards table. The possible status values are described in Leaderboard Statuses. A Delete button appears in this column when a leaderboard is eligible for deletion. After a leaderboard is available in Game Center, it can’t be deleted. Once you submit a leaderboard for review, you can’t delete it.

  1. In Apps, select the app you want to view.

  2. In the sidebar, click Game Center.

  3. In the Leaderboard section, hold the pointer over the leaderboard you want to delete, then click the delete button.

  4. Click Delete.

    5. Optionally, you can delete a leaderboard by selecting one from the Leaderboard section and clicking Delete this leaderboard on bottom left.
