Achievements

Achievement metadata identifies each achievement and its behavior in App Store Connect. Enter language-specific text for display, and monitor the status to see if the achievement is live and approved with the app.

Learn how to manage achievements.

Achievement properties

The main achievement properties in App Store Connect identify each achievement and how they behave.

Properties

Description

Reference Name

An internal name that you must provide for each achievement. This is the name you use if you search for the achievement within App Store Connect.

Achievement ID

A chosen alphanumeric identifier for your achievement. This ID is limited to 100 characters (assumes single-byte characters). Your Achievement ID is a permanent setting and therefore can’t be edited at a later date.

Point Value

The points that your achievement is worth. There is a maximum of 100 points per achievement and a maximum of 1000 points total for all achievements.

Hidden

Achievements marked as Hidden remain hidden in Game Center until a player has achieved them.

Achievable More Than Once

An indication of whether the player can earn the achievement multiple times.

Achievement language properties

Achievement language properties collect language-specific text used in achievement display. You must include these properties for at least one language.

Property

Description

Localization

The language in which you’d like this achievement to appear. There must be at least one language defined.

Display Name

The localized title of this achievement as you’d like it to appear in Game Center.

Earned Description

The description of your achievement as it appears to Game Center users after they earn it.

Pre-earned Description

The description of your achievement as it appears to Game Center users before they earn it.

Image

A localized image that represents the achievement. The image must be a .jpeg, .jpg, or .png file that’s 1024 x 1024 pixels, at least 72 ppi, and in the RGB color space. This property is required.

Achievement statuses

Achievement statuses indicate whether the achievement has been approved and made live with the app.

Status

Description

Prepare for Submission

You’ve created a Game Center component in App Store Connect but you're still preparing information for it.

Ready for Review

You’ve entered all of the required metadata for your Game Center component and indicated that you intend to submit it for review, but haven’t submitted it yet.

Learn how to submit Game Center components for review.

Waiting for Review

Apple received your submission, but hasn’t started the review.

While you're waiting for the review, you can remove your Game Center component from submission.

In Review

App Review is reviewing your Game Center component.

Accepted

This Game Center component has been accepted, but one or more items in your submission were rejected.

The component can only publish when all items in the submission are accepted. You can edit the rejected items and resubmit, or remove them from the submission to move forward with the currently accepted items.

Live

Your Game Center component has been accepted and is ready for distribution.

Rejected

Your Game Center component wasn't accepted. App Review notifies App Store Connect users with the Admin, App manager, or Developer role with the issues.

Learn how to reply to App Review messages.

Developer Rejected

You removed your Game Center component from review. When you’re ready, resubmit your component.
