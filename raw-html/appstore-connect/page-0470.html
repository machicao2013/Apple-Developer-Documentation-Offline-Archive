Delete testers from TestFlight

Testers deleted from TestFlight lose access to new builds but retain access to existing builds until they expire.

Learn how to manage testers with the App Store Connect API.

  1. In Apps, select the app you want to view.

  2. Click the TestFlight tab.

  3. In the sidebar, under Testers, click All.

  4. Add filters as needed, select the testers you want to delete, then click Delete above the table.

    Note: You can either select individual testers by selecting each checkbox next to the tester or select the checkbox in the header of the table to select all testers and then click Delete.

  5. In the dialog that appears, select the Delete checkbox and click Delete.

    Note: If a tester has a build installed, they'll have access to the build until it expires.

To view all deleted testers, visit the Testers tab, click Add Filter, click Status, select Deleted, and click Add.

Tip: To identify inactive testers, visit the Testers tab, click Add Filter, click Sessions, select None, and click Add. You can add additional filters, like Time Range, to identify testers who have been inactive for the last 7, 30, or 90 days.
