Test a beta version

Stop testing a build

Required role: Account Holder, Admin, or App Manager. View role permissions.

  1. In Apps, select the app you want to review.

  2. Click the TestFlight tab.

  3. In the sidebar, under Builds, click the platform.

  4. In the table on the right, in the Build column, click the app icon or build string for the build you want to expire.

  5. Click Expire Build.

The build no longer allows internal and external testers to install it.
