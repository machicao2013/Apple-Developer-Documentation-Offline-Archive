Manage leaderboard sets

Leaderboard sets organize several leaderboards into a single unit. For example, in a game that includes many levels, you could define a leaderboard set to organize the leaderboards for each level.

The limit to how many leaderboards you can create per app differs based on whether it’s with or without leaderboard sets:

Without Leaderboard sets:

  • Up to 100 leaderboards per app

With Leaderboard sets:

  • Up to 500 leaderboards total per app

    • Maximum 100 leaderboard sets per app

    • Maximum 100 leaderboards per set

You must have at least one leaderboard for your app before you can create a leaderboard set. After you add leaderboard sets to your app, all future leaderboards that you configure for the app must be included in a leaderboard set.

The process to start using leaderboard sets to organize your app’s leaderboards includes these steps:

  • Create the first leaderboard set.

  • Create additional leaderboard sets.

  • Add new leaderboards directly into leaderboard sets.

When you create the first leaderboard set, App Store Connect ensures that all existing leaderboards for the app are included in at least one leaderboard set.

Leaderboard sets and their leaderboard must be approved before they are available to players. Learn how to submit Game Center components for review.

Learn how to configure leaderboard sets with the Gamekit.

Required role: Account Holder, Admin, App Manager, Developer, or Marketing. View role permissions.

Merge leaderboards into sets

  1. In Apps, select the app you want to view.

  2. In the sidebar, click Game Center.

  3. In the Leaderboards section, select Move Leaderboards Into Sets from the ellipsis menu (...).

  4. Enter the Leaderboard Set Reference Name and Leaderboard Set ID, then click Continue.

  5. Click Add to Leaderboard Set to select which leaderboards you'd like to include in the set.

  6. Enter the Display Name Localization, then click Save.

  7. Under Leaderboard Set Localization, add one or more languages, then click Save.

    If this is the first leaderboard set you’re adding to an app, submit it for review.

You can add new leaderboard sets to your app to further customize how your leaderboards are displayed. Each set provides you with the ability to specify how a leaderboard is displayed within the set. You can put the same leaderboard into more than one set and configure a different display name for the leaderboard within each set.

Add a new leaderboard set

  1. In Apps, select the app you want to view.

  2. In the sidebar, click Game Center.

  3. In the Leaderboard Sets section, click the add button (+).

    Note: If group leaderboards exist, click the ellipsis menu (...) in the Group Leaderboards section and select Move Leaderboards Into Sets. Click Next.

  4. Enter the leaderboard set reference name and leaderboard set ID, then click Create.

    When you're ready, submit your leaderboard sets for review.

Add an existing leaderboard to a set

  1. In Apps, select the app you want to view.

  2. In the sidebar, click Game Center.

  3. In the Leaderboard Sets section, select the leaderboard set you want to add your existing leaderboard to.

  4. In the Leaderboards in Set section, click Add Leaderboard then select Add Existing Leaderboard.

    Note: If leaderboards exist, click the add button(+) in the Leaderboards in Set, then select Add Existing Leaderboard.

  5. Select the leaderboard you want to add.

  6. Click Save.

    If this is the first leaderboard or leaderboard set you’re adding to an app, submit it for review.

Add a new leaderboard to a set

  1. In Apps, select the app you want to view.

  2. In the sidebar, click Game Center.

  3. In the Leaderboards in Set section, click Add Leaderboard then select Add New Leaderboard.

    Note: If leaderboards exist, click the add button(+) in the Leaderboards in Set, then select Add New Leaderboard.

  4. Click the add button(+) in the Leaderboards in Set, then select Add New Leaderboard.

    If this is the first leaderboard or leaderboard set you’re adding to an app, submit it for review.

    5. Learn how to configure leaderboards in App Store Connect.

Delete leaderboard sets

Before deleting a leaderboard set, you must ensure that all leaderboards within the set have been added to another set. Only after the leaderboards have been added to a new set can you delete an existing set.

  1. In Apps, select the app you want to view.

  2. In the sidebar, click Game Center.

  3. In the Leaderboard Sets section, hold the pointer over the leaderboard set you want to delete, then click the Delete button.

  4. Click Save.

If your app contains 100 or fewer leaderboards, you can click the ellipsis menu (...) in the Leaderboard Sets section and click Remove All Leaderboards to remove all of your current leaderboards from their leaderboard sets and delete all of your current leaderboard sets.

Note: If your app contains more than 100 leaderboards, remove any excess leaderboards before you can use the Remove All Leaderboards option.

Optionally, you can delete a leaderboard set by selecting one from the Leaderboard sets section and clicking Delete this Leaderboard Set on bottom left.
